<p align="center">
      <h1 align="center"> 
        <a href="https://github.com/Crauzer/lol2gltf/wiki">How to use</a> 
        | 
        <a href="https://www.youtube.com/watch?v=XxSGk6SAcAM">Video Tutorial</a> 
      </h1>
</p>

* Features:
* * Skinned Mesh (skn, skl, anm) -> glTF
* * Map Geometry -> glTF
* * glTF -> SKN/SKL

* Upcoming features:
* * glTF -> ANM
